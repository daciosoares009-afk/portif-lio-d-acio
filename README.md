# portif-lio-d-acio# Portfólio - Dácio Soares

Portfólio pessoal desenvolvido para apresentar minhas habilidades como Desenvolvedor Full Stack em formação, meus projetos, trajetória de estudos e formas de contato.

## Sobre o projeto

Este site foi criado com HTML, CSS e JavaScript puro. A página conta com navegação responsiva, seção de apresentação, tecnologias, experiência, projetos e formulário de contato.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Tabler Icons
- Devicon

## Funcionalidades

- Menu responsivo para dispositivos móveis
- Scroll suave entre as seções
- Animações ao rolar a página
- Efeitos de hover nos cards e ícones
- Cards separados por área de conhecimento
- Ano dinâmico no rodapé
- Formulário de contato com validação básica

## Estrutura de arquivos

```text
portfolio/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Como executar

Abra o arquivo `index.html` diretamente no navegador.

Também é possível usar a extensão Live Server no VS Code para visualizar o projeto com recarregamento automático.

## Próximos ajustes

- Trocar os links genéricos do GitHub, LinkedIn e projetos pelos links reais
- Conectar o formulário a um serviço como Formspree, EmailJS ou backend próprio
- Adicionar imagens ou capturas de tela dos projetos
- Publicar o portfólio no GitHub Pages, Vercel ou Netlify
